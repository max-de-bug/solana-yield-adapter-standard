<Note>
This repository is a **reference implementation**. Adapters demonstrate the standard with local SPL vaults — production deployments require protocol-specific CPI.
</Note>

**Solana Yield Adapter Standard (SYAS)** is a minimal interface for yield on Solana: three instructions, one dispatcher, governance-gated registry, and pluggable adapters.

Use the **Overview** section in the sidebar to explore each topic, or jump to [Quick Start](/quickstart).
