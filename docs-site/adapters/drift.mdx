## Overview

The Drift adapter integrates with [Drift Protocol's](https://drift.trade) Insurance Fund. Users stake USDC to backstop the protocol and earn staking rewards.

## 13-Day Cooldown

The Drift Insurance Fund enforces a **13-day unstaking cooldown**:

1. User calls `withdraw()` → cooldown timer starts
2. 13 days must elapse before funds can be released
3. Adapter tracks `last_unstake_request` timestamp
4. Subsequent withdrawals check `elapsed >= 13 days`

```rust
pub const UNSTAKE_COOLDOWN_SECONDS: i64 = 13 * 24 * 60 * 60; // 1,123,200s
```

This is enforced on-chain and cannot be bypassed.

## Protocol-Specific Error Codes

| Code | Name | Description |
|---|---|---|
| `7000` | `CooldownNotElapsed` | 13-day unstaking period hasn't passed |
