## Overview

Drift's Insurance Fund lets stakers backstop the protocol and earn rewards. This reference adapter adds a **protocol-specific constraint** on top of the standard interface: withdrawals fail until a 13-day cooldown elapses after the first unstake request.

## 13-day cooldown

```rust
pub const UNSTAKE_COOLDOWN_SECONDS: i64 = 13 * 24 * 60 * 60; // 1,123,200s
```

<Steps>
  <Step title="User calls withdraw">
    First withdraw sets `last_unstake_request` on `AdapterPosition`.
  </Step>
  <Step title="Cooldown enforced">
    Subsequent withdraws check `clock.unix_timestamp - last_unstake_request >= UNSTAKE_COOLDOWN_SECONDS`.
  </Step>
  <Step title="Error on early exit">
    Returns `CooldownNotElapsed` (error code `7000`) if cooldown not met.
  </Step>
</Steps>

## Protocol-specific error

| Code | Name | Description |
|---|---|---|
| `7000` | `CooldownNotElapsed` | 13-day unstaking period has not passed |
