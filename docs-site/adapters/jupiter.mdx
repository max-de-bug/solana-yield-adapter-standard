## Overview

Jupiter's perpetuals liquidity pool (JLP) lets LPs earn trading fees from perp activity. This adapter names and structures accounts after that model while using the standard share-vault math from `yield-adapter-trait`.

## Instruction summary

| Instruction | Input | Effect |
|---|---|---|
| `deposit` | Underlying amount | Mint pool shares; transfer USDC to vault |
| `withdraw` | Receipt shares | Burn shares; return underlying |
| `current_value` | — | User's underlying position value |
