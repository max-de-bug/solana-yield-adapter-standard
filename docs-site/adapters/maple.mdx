## Overview

[syrupUSDC](https://maple.finance) is a yield-bearing SPL token on Solana mainnet that appreciates vs USDC as Maple's institutional lending pools generate returns. This adapter:

1. Implements the full three-instruction standard with syrupUSDC as the underlying
2. Holds real syrupUSDC tokens in its vault ATA — yield comes from the token's intrinsic appreciation
3. Requires no protocol CPI or time-based simulation (yield is embedded in the token itself)

## Instruction summary

| Instruction | Input | Effect |
|---|---|---|
| `deposit` | Underlying amount | Mint shares at current NAV |
| `withdraw` | Receipt shares | Burn shares at current NAV |
| `current_value` | — | User underlying value via share price |

## When to use this pattern

Teams building adapters for protocols that:

- Issue yield-bearing SPL tokens (no separate Solana program needed)
- Are EVM-primary with no live Solana program
- Want a lightweight adapter that treats the SPL token itself as the yield source

Unlike the earlier simulated-yield approach, this pattern holds real syrupUSDC and benefits from actual Maple pool returns through token appreciation.
