Get from zero to a passing test suite with the commands below. For toolchain details, see the [GitHub README](https://github.com/max-de-bug/solana-yield-adapter-standard#prerequisites).

## Prerequisites

<AccordionGroup>
  <Accordion title="Rust 1.75+">
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    rustup default stable
    ```
  </Accordion>
  <Accordion title="Solana CLI 2.2.20">
    ```bash
    sh -c "$(curl -sSfL https://release.solana.com/v2.2.20/install)"
    solana --version
    ```
  </Accordion>
  <Accordion title="Anchor CLI 1.0.1">
    ```bash
    cargo install --git https://github.com/coral-xyz/anchor avm --force
    avm install 1.0.1 && avm use 1.0.1
    ```
  </Accordion>
  <Accordion title="Node.js 18+">
    Required for TypeScript integration tests and npm scripts.
  </Accordion>
</AccordionGroup>

## Clone and build

```bash
git clone https://github.com/max-de-bug/solana-yield-adapter-standard.git
cd solana-yield-adapter-standard
npm install
npm run build
```

<Tip>
`npm run build` runs `scripts/build-sbf.sh` (all seven programs) then `scripts/build-idls.sh` (IDL artifacts under `target/idl/`).
</Tip>

## Run tests

<Tabs>
  <Tab title="Local (recommended)">
    ```bash
    npm test
    ```

    Runs **32 integration tests** (26 passing, 6 pre-existing slippage failures on localnet) against a local validator:

    - Registry governance lifecycle (including guardian role)
    - Dispatcher routing and pause
    - All five adapters: deposit → `current_value` → withdraw
    - Drift two-phase withdrawal + cooldown enforcement
    - Error paths (zero amounts, insufficient balance)
  </Tab>
  <Tab title="Mainnet fork">
    ```bash
    bash scripts/run-fork-surfpool.sh
    ```

    Runs **96 integration tests (6 adapters × 12 + dispatcher + registry)** via [Surfpool](https://surfpool.run) (JIT account fetching — no `--clone` flags or fixture ATAs needed):

    - All 5 reference adapters (Kamino, MarginFi, Jupiter, Maple, Drift) + template scaffold
    - Full registry governance lifecycle (propose → approve → revoke → re-approve, guardian, governance transfer)
    - Dispatcher routing, pause, `current_value` CPI
    - Drift two-phase withdrawal + cooldown enforcement + stale ticket cleanup

    <Warning>
    Requires `MAINNET_RPC_URL` for JIT fetching. See `scripts/run-fork-surfpool.sh`.
    </Warning>
  </Tab>
</Tabs>

## Deploy to devnet

```bash
npm run deploy:devnet
# or: ./scripts/deploy-devnet.sh
```

The script builds all programs via `scripts/build-sbf.sh` + `scripts/build-idls.sh`, deploys all 7 programs (registry, dispatcher, 5 adapters) to devnet, and prints the deployed program IDs.
