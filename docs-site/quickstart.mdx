# Quick Start

## Prerequisites

<Steps>
  <Step title="Install Rust">
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```
  </Step>
  <Step title="Install Solana CLI">
    ```bash
    sh -c "$(curl -sSfL https://release.solana.com/v2.2.20/install)"
    ```
  </Step>
  <Step title="Install Anchor CLI">
    ```bash
    cargo install --git https://github.com/coral-xyz/anchor avm --force
    avm install 1.0.1
    avm use 1.0.1
    agave-install init 2.2.20
    ```
  </Step>
</Steps>

## Build

```bash
git clone https://github.com/your-org/solana-yield-adapter-standard.git
cd solana-yield-adapter-standard
npm install
npm run build
```

## Test

```bash
# Unit tests
anchor test

# Mainnet-fork tests
./scripts/run-mainnet-fork-tests.sh
```

## Deploy to Devnet

```bash
./scripts/deploy-devnet.sh
```

<Info>
The deploy script generates keypairs, builds programs, deploys to devnet, and outputs the program IDs to update in `Anchor.toml`.
</Info>
