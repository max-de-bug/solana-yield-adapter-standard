---
title: Error Codes
description: Complete error code reference across trait, dispatcher, registry, and adapters
---

Errors are namespaced by numeric range so integrators can branch on category without parsing program-specific strings.

## Standard adapter errors (`6000–6099`)

Defined in `crates/yield-adapter-trait`. All adapters reuse these.

| Code | Name | When |
|---|---|---|
| 6000 | `ZeroDepositAmount` | `deposit(0)` |
| 6001 | `ZeroWithdrawAmount` | `withdraw(0)` |
| 6002 | `InsufficientReceiptBalance` | Burn exceeds position balance |
| 6003 | `AdapterNotActive` | Vault status rejects the operation (paused, deposits paused, or deprecated) |
| 6004 | `MintMismatch` | Token mint ≠ vault underlying mint |
| 6005 | `AdapterProgramMismatch` | Wrong adapter program in accounts |
| 6006 | `ArithmeticOverflow` | Share math overflow |
| 6007 | `ProtocolCpiError` | Reserved for production CPI failures |
| 6008 | `Unauthorized` | Signer lacks authority |
| 6009 | `PositionNotInitialized` | `AdapterPosition` missing |
| 6010 | `InvalidMetadata` | Bad adapter metadata |
| 6011 | `VaultDeprecated` | Vault permanently retired |
| 6012 | `SlippageExceeded` | Shares or underlying below user-specified minimum (slippage protection) |

## Dispatcher errors (`6100–6199`)

| Code | Name | When |
|---|---|---|
| 6100 | `DispatcherPaused` | `is_paused == true` |
| 6101 | `AdapterNotApproved` | Registry status ≠ Approved |
| 6102 | `ZeroAmount` | Zero deposit/withdraw amount |
| 6103 | `Unauthorized` | Non-authority calls admin ix |
| 6104 | `AdapterCpiError` | Adapter CPI returned error |
| 6105 | `RegistryMismatch` | Wrong registry program ID |

## Registry errors (`6200–6299`)

| Code | Name | When |
|---|---|---|
| 6200 | `Unauthorized` | Non-governance action |
| 6201 | `NameTooLong` | Name &gt; 32 chars |
| 6202 | `UriTooLong` | URI &gt; 200 chars |
| 6203 | `InvalidStatus` | Wrong status for operation |
| 6204 | `AlreadyRegistered` | Duplicate adapter entry |
| 6205 | `NoPendingTransfer` | No pending governance handoff |
| 6206 | `NotPendingAuthority` | Signer is not pending authority |

## Protocol-specific errors (`7000+`)

Reserve this range for adapter-specific constraints. Document each code in the adapter's docs page.

| Code | Adapter | Name | When |
|---|---|---|---|
| 7000 | Drift | `CooldownNotElapsed` | Withdraw before 13-day cooldown |

## Handling in clients

```typescript
try {
  await program.methods.deposit(amount).accounts({...}).rpc();
} catch (e) {
  if (e.error?.errorCode?.code === "AdapterNotApproved") {
    // Adapter revoked or never approved
  }
}
```
