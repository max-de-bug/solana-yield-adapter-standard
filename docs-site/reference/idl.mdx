# IDL Reference

Each program's IDL is auto-generated by Anchor during `anchor build` and saved to `target/idl/`.

## Programs

| Program | IDL Path |
|---|---|
| Yield Dispatcher | `target/idl/yield_dispatcher.json` |
| Adapter Registry | `target/idl/adapter_registry.json` |
| Kamino Adapter | `target/idl/adapter_kamino.json` |
| MarginFi Adapter | `target/idl/adapter_marginfi.json` |
| Jupiter Adapter | `target/idl/adapter_jupiter.json` |
| Maple Adapter | `target/idl/adapter_maple.json` |
| Drift Adapter | `target/idl/adapter_drift.json` |

## Using the IDL

### TypeScript

```typescript
import idl from "../target/idl/adapter_registry.json";
import { Program } from "@coral-xyz/anchor";

const program = new Program(idl, provider);
```

### Rust (CPI)

```rust
// Add the program as a dependency with the `cpi` feature
[dependencies]
adapter-registry = { path = "../adapter-registry", features = ["cpi"] }
```

## Generating Fresh IDLs

```bash
anchor build
# IDLs are output to target/idl/
```
