# Testing

## Unit Tests (Localnet)

```bash
anchor test
```

Runs against a local Solana test validator. Tests cover:
- Registry governance lifecycle
- Dispatcher routing logic
- Individual adapter deposit/withdraw/value flows
- Error cases and edge conditions

## Mainnet-Fork Tests

```bash
./scripts/run-mainnet-fork-tests.sh
```

The script:
1. Builds all Anchor programs
2. Starts `solana-test-validator` with cloned mainnet programs
3. Deploys your programs to the forked environment
4. Runs integration tests against real on-chain state

### Writing Fork Tests

```typescript
describe("kamino-fork", () => {
  it("deposits into live Kamino market", async () => {
    // Uses real Kamino program state cloned from mainnet
    // Test against actual reserve accounts and pool state
  });
});
```

<Warning>
Mainnet-fork tests require a stable internet connection to clone program state. They may take longer to start than local tests.
</Warning>
