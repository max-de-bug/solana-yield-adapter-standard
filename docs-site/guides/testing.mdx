The test suite validates registry governance, dispatcher routing, and all five adapters end-to-end. Run locally before every PR.

## Test matrix

| Command | Tests | Environment | Duration |
|---|---|---|---|---|
| `npm test` | 32 (26 passing, 6 pre-existing slippage failures on localnet) | Local validator | ~2 min |
| `MAINNET_FORK=1` (see below) | 31 | Cloned mainnet programs | ~5–15 min (first run) |

<Tip>
`npm test` uses `anchor test --validator legacy --skip-build`. Run `npm run build` first if programs changed.
</Tip>

## Local tests

```bash
npm run build
npm test
```

### Coverage

| Suite | File | What it verifies |
|---|---|---|---|
| Registry | `tests/registry.test.ts` | propose → approve → revoke → set guardian → governance transfer |
| Dispatcher | `tests/dispatcher.test.ts` | Routing, pause, registry rejection |
| Adapters | `tests/adapters/*.test.ts` | deposit → `current_value` → withdraw per adapter |
| Drift | `tests/adapters/drift.test.ts` | Two-phase withdraw, cooldown enforcement, slippage at request time |
| Errors | Across suites | Zero amounts, insufficient balance, revoked adapters |

### Test helpers

Reusable wiring lives in `tests/helpers/`:

- `adapter.ts` — adapter deposit/withdraw/value account builders
- Shared token mints, PDAs, and program clients

Copy these patterns when adding a sixth adapter.

## Mainnet-fork tests

```bash
./scripts/run-fork-solana-test-validator.sh
```

The script (`scripts/run-fork-solana-test-validator.sh`):

1. Builds all programs and IDLs
2. Starts `solana-test-validator` with `--clone` for mainnet programs and `--account` for fixture ATAs
3. Deploys your programs to the fork
4. Runs `MAINNET_FORK=1 anchor test` against cloned state

## Writing new tests

Minimum cases for any adapter:

```typescript
it("deposit → current_value → withdraw", async () => {
  await deposit(1_000_000);
  const value = await currentValue();
  expect(value).to.be.gt(0);
  await withdraw(shares);
});

it("rejects zero deposit", async () => {
  await expect(deposit(0)).to.be.rejected;
});
```
